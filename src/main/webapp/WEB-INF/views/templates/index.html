Программа по определению величины таможенных сборов

Данный курсовой проект был реализован на примере Таможенного калькулятора по расчету стоимости ввозимых автомобилей на территорию Беларуси

В курсовом проекте использовались следующие технологии: Spring Security, Spring MVC, Spring Data, MySQL, Thymeleaf, Bootstrap, Spring, Hibernate
